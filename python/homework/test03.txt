리눅스를 활용하면서 기본적으로 많은 셀 명령어들을 활용하게 된다. 관련 명령어와 사용법 등을 조사하여 10개의 명령어를 정리하여 제출하시오.


->
  1.ls : 파일 목록을 보여주는 명령어
      -a 디렉토리 안에 있는 모든 파일과 디렉토리를 보여줌.
      -l 파일에 대한 정보를 자세하게 보여줍니다. 파일의 사용권한, 소유자, 그룹, 크기, 날짜 등을 자세하게 보여줌.
      -al
  2.touch : 파일의 시간정보를 현재 시간으로 변경. 만약 파일이 없다면 빈파일을 생성
  3.cat : 파일을 터미널 화면에 뿌려줌.(복수의 파일을 입력가능), >를 통해 여러 파일의 내용을 합칠수 있음.
  4.chmod :  파일의 접근 권한을 8인수숫자 3개를 통해서 바꿀 수 있음.
  5.cp : copy의 약자로 파일을 복사.
  6.ps : 프로세서들의 상태를 출력.
      -l :자세한 형태의 정보를 출력
      -e :모든 프로세스를 보여줌.
  7.kill : 프로세스를 종료.
  8.pwd : 현재 디렉토리의 위치.
  9.cd : 디렉토리 이동.
  10.history : 지금까지 실행되었던 명령어들의 리스트를 출력
